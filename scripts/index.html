Skill API Reference

Call ai_edge_gallery_get_result with a JSON string before each session to inject context.

```
get_context — inject all memories before a chat 
// Paste this as the skill tool call in your agent:
```

{
“action”: “get_context”
}
// Returns formatted memory block for the AI’s context window.
```
save_memory — store a new memory 
{
```

“action”:   “save_memory”,
“title”:   “My name”,
“content”: “My name is Alex and I prefer concise answers.”,
“tags”:    “identity, preferences”  // optional
}
```
search_memories — find memories by keyword 
{
```

“action”: “search_memories”,
“query”:  “coding”
}
```
delete_memory — remove by id 
{
```

“action”: “delete_memory”,
“id”:     ””
}
```
wipe_memories — erase everything 
{
```

“action”: “wipe_memories”
}
// ⚠ Irreversible.
